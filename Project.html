Build Your First AI Chatbot

In this project, we will build a Simple AI Chatbot using Python that can respond to user inputs conversationally and perform basic tasks.

Step 1: Setting Up

Ensure you have Python installed on your system. Use an editor like VS Code, PyCharm, or IDLE for writing the code.

Step 2: Writing the Code

import datetime

def chatbot():
    print("Hello! I am your friendly AI Chatbot.")
    print("You can ask me simple questions like:\n- 'What is your name?'\n- 'What time is it?'\n- 'How are you?'")
    print("Type 'exit' to end the conversation.\n")

    while True:
        # Get user input
        user_input = input("You: ").lower()
        
        # Respond to greetings
        if user_input in ["hi", "hello", "hey"]:
            print("Chatbot: Hello! How can I help you today?")
        
        # Respond to name question
        elif "your name" in user_input:
            print("Chatbot: My name is ChatBot 101, your friendly assistant!")
        
        # Respond to time query
        elif "time" in user_input:
            current_time = datetime.datetime.now().strftime("%H:%M:%S")
            print(f"Chatbot: The current time is {current_time}.")
        
        # Respond to "how are you?"
        elif "how are you" in user_input:
            print("Chatbot: I'm just a bunch of code, but I'm feeling chatty!")
        
        # Exit the program
        elif user_input == "exit":
            print("Chatbot: Goodbye! Have a great day!")
            break
        
        # Handle unrecognized inputs
        else:
            print("Chatbot: I'm sorry, I don't understand that. Can you try asking something else?")

# Run the chatbot
chatbot()

Step 3: Running the Project

Save the code in a file named chatbot.py and run it using the following command:

python chatbot.py

Step 4: Interacting with the Chatbot

Once the chatbot is running, try asking it the following:

  • "Hi" or "Hello"
  • "What is your name?"
  • "What time is it?"
  • "How are you?"
  • Type "exit" to stop the chatbot.
